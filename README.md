# Author Rezognition ML project

### Data processing

```bash
 $ python process_data.py [-v] [--full]
```

You can set target file inside process_data.py

flags:
-v: Verbose mode
--full: Run with 10k records instead of 100

---

### Model training

```bash
 $ python train.py
```

You can set data source file inside train.py
